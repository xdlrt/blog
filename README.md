# 一颗小树

In progress to be a product maker.

## 技术栈

- Astro 3.x
- Typescript
- TailwindCSS
- Playwright

## 环境要求

- Node.js 18.x
- NPM 9.x

中国大陆建议使用 CNPM：

```bash
npm install cnpm -g --registry=https://registry.npmmirror.com
```

## 依赖安装

```bash
npm i
```

## 本地开发

```bash
npm run dev
```

## E2E 测试

```bash
npm run test:ui
```

PlayWright UI Mode

```bash
npm run test:ui:debug
```
